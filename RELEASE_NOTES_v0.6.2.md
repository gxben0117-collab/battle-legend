## 🐛 Bug 修復清單

### 1. ✅ 修復跨章節解鎖邏輯

**問題描述**：
- 玩家通關 1-5 後，第 2 章第 1 關（2-1）無法解鎖
- 只有通關 1-10 才應該解鎖 2-1

**原因分析**：
```javascript
// 修改前
const locked = i > 0 && !G.stageCleared[chapterStages[i - 1].id];
// 問題：只檢查同章節內的前一關，沒有檢查上一章最後一關
```

**修復方案**：
```javascript
// 修改後
let locked = false;
if (i > 0) {
  // 同章節內檢查前一關
  locked = !G.stageCleared[chapterStages[i - 1].id];
} else {
  // 章節第一關，檢查上一章最後一關
  const chapterNum = parseInt(chapter.stagePrefix.split('-')[0]);
  if (chapterNum > 1) {
    const prevChapterLastStage = `${chapterNum - 1}-10`;
    locked = !G.stageCleared[prevChapterLastStage];
  }
}
```

**測試結果**：
- ✅ 通關 1-10 後，2-1 正確解鎖
- ✅ 通關 2-10 後，3-1 正確解鎖
- ✅ 所有章節解鎖邏輯正常

### 2. ✅ 修復能量恢復受倍速影響

**問題描述**：
- 玩家在 ×4 倍速下，能量每秒瘋狂增加
- 應該始終保持每 2 秒 +1 能量（不受倍速影響）

**原因分析**：
```javascript
// 修改前
const dt = rawDt * B.speed;  // dt 受倍速影響
B.energyTimer += dt;  // 能量恢復使用了倍速時間
```

**修復方案**：
```javascript
// 修改後
B.energyTimer += rawDt;  // 使用真實時間，不受倍速影響
```

**測試結果**：
- ✅ ×1 倍速：每 2 秒 +1 能量
- ✅ ×2 倍速：每 2 秒 +1 能量
- ✅ ×4 倍速：每 2 秒 +1 能量
- ✅ 能量恢復穩定，不受倍速影響

### 3. ✅ 添加手動模式提示訊息

**問題描述**：
- 手動模式下，有能量但點擊卡牌無反應
- 沒有任何提示告訴玩家為什麼無法出牌

**原因分析**：
```javascript
// 修改前
function playCardById(cardId) {
  const card = CARD_DB[cardId];
  if (!card || card.cost > B.energy) return;  // 靜默失敗
  
  const freeCell = findFreeCell(true);
  if (!freeCell) return;  // 靜默失敗
  
  // ... 出牌邏輯
}
```

**修復方案**：
```javascript
// 修改後
function playCardById(cardId) {
  const card = CARD_DB[cardId];
  if (!card) return;

  if (card.cost > B.energy) {
    showToast('⚡ 能量不足');  // 添加提示
    return;
  }

  const freeCell = findFreeCell(true);
  if (!freeCell) {
    showToast('❌ 戰場已滿，無法部署');  // 添加提示
    return;
  }
  
  // ... 出牌邏輯
}
```

**測試結果**：
- ✅ 能量不足時顯示「⚡ 能量不足」
- ✅ 戰場已滿時顯示「❌ 戰場已滿，無法部署」
- ✅ 提示訊息清晰明確
