# 🎮 戰棋傳說 (Battle Legend)

一款基於 HTML5 的策略卡牌戰棋遊戲，採用單文件架構，無需編譯即可運行。

## 🌟 遊戲特色

### 核心玩法
- **9×9 戰場**：戰術空間更大，策略更豐富
- **卡牌召喚**：15+ 種獨特單位卡牌
- **自動/手動模式**：靈活切換戰鬥方式
- **倍速控制**：1×/2×/3× 戰鬥速度

### 遊戲系統
- **關卡系統**：3 章節 × 10 關卡 + BOSS 副本
- **牌組編輯**：30 張卡牌自由組合
- **裝備系統**：武器、護甲、飾品強化
- **羈絆系統**：特定組合觸發額外效果
- **商店系統**：購買卡牌和裝備

## 🎯 遊戲操作

### 戰鬥畫面
- **左側**：手牌區（顯示可用卡牌）
- **中間**：9×9 戰場（單位自動行動）
- **右側**：
  - 能量顯示（10/10 格式）
  - 🤖 自動 / 👆 手動切換
  - 1×/2×/3× 倍速控制
  - 🏳️ 撤退按鈕

### 基本流程
1. **能量累積**：每 2 秒恢復 1 點能量
2. **出牌**：
   - 自動模式：AI 自動出牌
   - 手動模式：點擊手牌出牌
3. **單位行動**：
   - 自動移動到敵人附近
   - 自動攻擊範圍內的敵人
   - 自動釋放技能
4. **勝利條件**：摧毀敵方主堡

## 📊 技術架構

### 單文件設計
- **HTML + CSS + JavaScript**：全部代碼在 `index.html` 中
- **無外部依賴**：純原生 Web 技術
- **localStorage 存檔**：自動保存遊戲進度

### 核心模塊
```
├── 數據層
│   ├── CARD_DB (卡牌數據庫)
│   ├── STAGES (關卡數據)
│   ├── BOSS_RAIDS (BOSS 副本)
│   └── EQUIPMENT_DB (裝備數據庫)
├── 狀態管理
│   ├── G (全局遊戲狀態)
│   └── B (戰鬥狀態)
├── 渲染系統
│   ├── 戰場渲染
│   ├── 單位渲染
│   ├── UI 渲染
│   └── 特效渲染
└── 邏輯系統
    ├── 戰鬥循環
    ├── AI 系統
    ├── 技能系統
    └── 羈絆系統
```
