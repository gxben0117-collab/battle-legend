## ⚙️ 自定義配置

### 修改測試次數
編輯 `playwright-test.js`，找到：
```javascript
const CONFIG = {
  rounds: 10,  // 改成你想要的次數（例如 100）
  // ...
};
```

### 修改倍速
```javascript
const CONFIG = {
  speed: 3,  // 改成 1, 2, 或 3
  // ...
};
```

### 無頭模式（不顯示瀏覽器）
```javascript
const CONFIG = {
  headless: true,  // true = 背景執行，false = 顯示瀏覽器
  // ...
};
```

### 測試不同關卡
修改腳本中的：
```javascript
// 找到這一行
const stage1Button = await page.$('text=1-1');

// 改成
const stage1Button = await page.$('text=2-5');  // 測試 2-5 關
```

## 🐛 常見問題

### Q: npm install 失敗
```bash
# 清除緩存重試
npm cache clean --force
npm install
```

### Q: 瀏覽器無法啟動
```bash
# 重新安裝瀏覽器
npm run install-browser
```

### Q: 測試卡住不動
- 檢查網路連線
- 檢查遊戲網址是否正確
- 按 Ctrl+C 停止測試

### Q: 想提前停止測試
```bash
按 Ctrl + C
```

## 🚀 進階用法

### 測試所有關卡
創建 `test-all-stages.js`：
```javascript
const { chromium } = require('playwright');

async function testAllStages() {
  for (let chapter = 1; chapter <= 3; chapter++) {
    for (let stage = 1; stage <= 10; stage++) {
      console.log(`\n測試關卡: ${chapter}-${stage}`);
      // 運行測試...
    }
  }
}

testAllStages();
```

### 截圖記錄
在腳本中添加：
```javascript
await page.screenshot({ path: `test-${testNum}.png` });
```

### 錄影記錄
```javascript
const context = await browser.newContext({
  recordVideo: { dir: 'videos/' }
});
```
