## 🎯 遊戲概述

**戰棋傳說 (Battle Legend)** 是一款融合自走棋、卡牌收集和戰術策略的單機遊戲。

### 核心玩法
- **ATB 戰鬥系統**：每個單位有獨立的行動條
- **卡牌收集**：30 張卡牌收集與組牌
- **羈絆系統**：種族/職業羈絆觸發強力效果
- **裝備系統**：武器/護甲/飾品永久強化
- **BOSS 挑戰**：5 個巨型 2×2 BOSS

### 目標玩家
- 喜歡策略遊戲的玩家
- 卡牌收集愛好者
- 自走棋玩家
- 休閒策略遊戲玩家

## ⚔️ 核心系統

### 1. ATB 戰鬥系統

**Active Time Battle（主動時間戰鬥）**

```javascript
// ATB 計算公式
atb += deltaTime * (unit.spd / 100)
當 atb >= maxAtb (1000) 時，單位行動
```

**特點**：
- 速度越高，行動越頻繁
- 實時戰鬥，無回合等待
- 可調整倍速（×1/×2/×4）

### 2. 物魔分離系統

**四維屬性**：
- **物理攻擊（patk）**：物理傷害輸出
- **魔法攻擊（matk）**：魔法傷害輸出
- **物理防禦（pdef）**：減免物理傷害
- **魔法防禦（mdef）**：減免魔法傷害

**傷害計算**：
```javascript
// 物理傷害
damage = max(1, attacker.patk - target.pdef * 0.5)

// 魔法傷害
damage = max(1, attacker.matk - target.mdef * 0.5)

// 暴擊傷害
if (crit) damage *= (critDmg / 100)
```

### 3. 羈絆系統

**7 種羈絆效果**

| 羈絆類型 | 激活條件 | 效果 |
|---------|----------|------|
| **人類** | 3/6/9 個 | 全體物攻 +10%/20%/35% |
| **精靈** | 2/4/6 個 | 全體速度 +15%/30%/50% |
| **獸人** | 2/4 個 | 全體物攻 +15%/30% |
| **亡靈** | 2/4 個 | 全體魔攻 +15%/30% |
| **龍族** | 1 個 | 全體物防+魔防 +20% |
| **戰士** | 2/4/6 個 | 全體生命 +15%/30%/50% |
| **法師** | 2/4/6 個 | 全體魔攻 +10%/20%/35% |

### 4. 能量系統

- **初始能量**：0
- **最大能量**：10
- **回復速度**：每 2 秒 +1
- **消耗**：出牌消耗 1-5 能量

### 5. 手動/自動模式

**自動模式**：
- 系統自動選擇最高費用可出牌
- 適合掛機戰鬥

**手動模式**：
- 玩家點擊手牌出戰
- 可選擇出牌時機
- 更高策略性
