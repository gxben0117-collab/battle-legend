## 🐛 問題詳情

### 問題 1：地圖顯示錯誤 ❌ → ✅

**症狀**：
- 戰場仍然顯示為 10×10 格子
- 卡片位置錯亂
- 主堡位置不正確

**原因**：
```javascript
// 錯誤代碼（setupBattlefield 函數）
for (let row = 0; row < 10; row++) {
  for (let col = 0; col < 10; col++) {
    // 仍然繪製 10×10 格子
  }
}
```

**修復**：
```javascript
// v0.7.1：正確的 9×7 戰場
for (let row = 0; row < 7; row++) {
  for (let col = 0; col < 9; col++) {
    // 繪製 9×7 格子
    // 我方陣地：x=3~5, y=5~6（下方）
    // 敵方陣地：x=3~5, y=0~1（上方）
    // 主堡：(4,6) 和 (4,0)
  }
}
```

**結果**：
- ✅ 戰場正確顯示為 9×7
- ✅ 主堡在上下中央
- ✅ 陣地正確標示

---

### 問題 2：血量顯示為百分比 ❌ → ✅

**症狀**：
- 單位血量顯示為「100%」、「57%」
- 無法看到真實血量數值

**原因**：
```javascript
// v0.3：舊版本使用百分比
hpTextEl.textContent = pct + '%';
```

**修復**：
```javascript
// v0.7.1：改為真實數值
hpTextEl.textContent = `${Math.round(u.hp)}/${u.maxHp}`;

// 初始化時
<span class="unit-hp-text">${Math.round(u.hp)}/${u.maxHp}</span>
```

**結果**：
- ✅ 顯示真實血量「75/100」
- ✅ 更清楚了解單位狀態
- ✅ 低血量時仍然變紅色

---

### 問題 3：攻擊特效簡化 ✅

**變更前**：
```javascript
// 複雜的判斷邏輯
if (attacker.range <= 1) {
  showMeleeSlash();  // 近戰揮刀
} else if (attacker.dmgType === 'physical') {
  showArrowEffect();  // 物理遠程：箭矢
} else {
  showMagicBolt();    // 魔法遠程：火球
}
```

**變更後**：
```javascript
// v0.7.1：簡化邏輯
if (attacker.range <= 1) {
  showMeleeSlash();   // 近戰：統一揮刀
} else {
  showMagicBolt();    // 遠程：統一光球
}
```

**優點**：
- ✅ 所有近戰都有揮刀特效 ⚔️
- ✅ 所有遠程都有光球特效 ✨
- ✅ 代碼更簡潔
- ✅ 特效更一致
