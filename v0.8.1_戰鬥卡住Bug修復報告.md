# 🐛 戰棋傳說 v0.8.1 - 戰鬥卡住 Bug 修復報告

**問題時間**：2025-06-11  
**版本**：v0.8.1  
**問題描述**：戰鬥在 12-20 秒後會卡住不動  
**狀態**：✅ 已修復並部署

---

## 🔍 問題分析

### 用戶報告
```
症狀：戰鬥進行 12-20 秒後完全卡住
表現：單位不動、能量不增、計時器停止
頻率：每次遊戲都會發生
```

### 可能原因
1. **JavaScript 錯誤未捕獲**：導致戰鬥循環中斷
2. **DOM 元素丟失**：getElementById 返回 null 導致崩潰
3. **Canvas 獲取失敗**：getCtx() 在 canvas 不存在時報錯
4. **無限循環或死鎖**：某個函數卡住不返回
5. **內存洩漏**：導致瀏覽器凍結

### 診斷發現

#### 問題 1：戰鬥循環無錯誤處理
```javascript
// 原代碼（有問題）
function loop(now) {
  if (!B || !B.running) return;
  // ... 所有邏輯 ...
  B.animFrameId = requestAnimationFrame(loop);
}
```

**問題**：
- 任何錯誤都會中斷循環
- 沒有錯誤提示
- 無法定位問題

#### 問題 2：getCtx() 不安全
```javascript
// 原代碼（有問題）
function getCtx() {
  return getCanvas().getContext('2d');
}
```

**問題**：
- 如果 canvas 不存在，直接報錯
- 每次調用都重新獲取 canvas
- 沒有 null 檢查

#### 問題 3：renderCanvas 缺少檢查
```javascript
// 原代碼（有問題）
function renderCanvas(dt) {
  const canvas = getCanvas();
  const ctx = getCtx();
  ctx.clearRect(0, 0, canvas.width, canvas.height); // 如果 ctx 是 null 會報錯
}
```

#### 問題 4：updateUnitBar 不檢查單位
```javascript
// 原代碼（有問題）
function updateUnitBar(u) {
  const hpEl = document.getElementById('uhp-' + u.uid); // 如果 u 是 undefined 會報錯
}
```

#### 問題 5：renderEnergy 不檢查 B
```javascript
// 原代碼（有問題）
function renderEnergy() {
  const energyEl = document.getElementById('energy');
  energyEl.textContent = `${B.energy}/${B.maxEnergy}`; // 如果 B 不存在會報錯
}
```

## 🔧 修復方案

### 修復 1：戰鬥循環添加 try-catch

#### 修復前
```javascript
function loop(now) {
  if (!B || !B.running) return;
  // ... 邏輯 ...
  B.animFrameId = requestAnimationFrame(loop);
}
```

#### 修復後
```javascript
function loop(now) {
  try {
    if (!B || !B.running) return;
    // ... 邏輯 ...
    B.animFrameId = requestAnimationFrame(loop);
  } catch (error) {
    // v0.8.1：捕獲戰鬥循環錯誤
    console.error('戰鬥循環錯誤:', error);
    console.error('錯誤堆疊:', error.stack);
    alert('戰鬥發生錯誤: ' + error.message + '\n請按 F12 查看控制台');
    B.running = false;
  }
}
```

**效果**：
- ✅ 捕獲所有錯誤
- ✅ 輸出詳細錯誤信息
- ✅ 提示用戶查看控制台
- ✅ 優雅停止戰鬥

---

### 修復 2：getCanvas/getCtx 添加檢查

#### 修復前
```javascript
function getCanvas() {
  return document.getElementById('battleCanvas');
}

function getCtx() {
  return getCanvas().getContext('2d');
}
```

#### 修復後
```javascript
function getCanvas() {
  const canvas = document.getElementById('battleCanvas');
  if (!canvas) {
    console.error('Canvas 元素不存在');
  }
  return canvas;
}

function getCtx() {
  const canvas = getCanvas();
  if (!canvas) return null; // v0.8.1：安全返回
  return canvas.getContext('2d');
}
```

**效果**：
- ✅ 返回 null 而不是報錯
- ✅ 輸出錯誤日誌
- ✅ 後續代碼可以檢查 null

---

### 修復 3：renderCanvas 添加安全檢查

#### 修復前
```javascript
function renderCanvas(dt) {
  if (!B) return;
  const canvas = getCanvas();
  const ctx = getCtx();
  ctx.clearRect(0, 0, canvas.width, canvas.height);
  // ...
}
```

#### 修復後
```javascript
function renderCanvas(dt) {
  if (!B) return;
  const canvas = getCanvas();
  if (!canvas) return; // v0.8.1：安全檢查

  const ctx = getCtx();
  if (!ctx) return; // v0.8.1：安全檢查

  ctx.clearRect(0, 0, canvas.width, canvas.height);
  // ...
}
```

**效果**：
- ✅ 檢查 canvas 是否存在
- ✅ 檢查 ctx 是否存在
- ✅ 避免 null 引用錯誤
