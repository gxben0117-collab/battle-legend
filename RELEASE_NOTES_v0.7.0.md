## 📊 詳細變更

### 1. 地圖系統重構

**v0.6.2（舊）**：
- 地圖大小：10×10
- 主堡位置：我方 (0,9)、敵人 (9,0)
- 開場陣地：我方左下 3×3、敵人右上 3×3

**v0.7.0（新）**：
- 地圖大小：9×7（更緊湊）
- 主堡位置：我方 (4,6) 下方中央、敵人 (4,0) 上方中央
- 開場陣地：我方下方 2×3、敵人上方 2×3
- 格子大小：64px → 72px（更大更清晰）

**優點**：
- ✅ 戰鬥更緊湊快速
- ✅ 視覺更集中
- ✅ 佈局更合理（上下對稱）

---

### 2. 攻擊特效系統

**新增 4 種攻擊特效**：

#### ⚔️ 近戰揮刀特效
```javascript
// 判斷：range <= 1
showMeleeSlash(fromX, fromY, toX, toY);
// 效果：劍型圖示 + 旋轉 + 淡入淡出動畫
```

#### 🏹 弓箭射擊特效
```javascript
// 判斷：range > 1 && dmgType === 'physical'
showArrowEffect(fromX, fromY, toX, toY);
// 效果：箭矢圖示 + 飛行軌跡 + 自動轉向
```

#### 🔥 魔法火球特效
```javascript
// 判斷：range > 1 && dmgType === 'magic'
showMagicBolt(fromX, fromY, toX, toY);
// 效果：火球圖示 + 拋物線飛行 + 爆炸縮放
```

#### 💫 CSS 動畫
```css
@keyframes slash-fade {
  0% { opacity: 0; transform: scale(0.5) rotate(45deg); }
  50% { opacity: 1; transform: scale(1.2) rotate(45deg); }
  100% { opacity: 0; transform: scale(0.8) rotate(45deg); }
}
```

**效果展示**：
- 近戰單位攻擊：⚔️ 揮刀動畫
- 弓箭手攻擊：🏹 箭矢飛向敵人
- 法師攻擊：🔥 火球飛向敵人
- 攻擊時長：0.3-0.45 秒

---

### 3. 戰鬥節奏優化

#### 提高攻擊頻率
```javascript
// 修改前
u.atb += u.spd * dt / 1000 * 2;  // ATB ×2

// 修改後
u.atb += u.spd * dt / 1000 * 3;  // ATB ×3
```

**效果**：
- ✅ 單位行動更頻繁
- ✅ 戰鬥節奏更快
- ✅ 更有動作感

#### 提高血量 50%
```javascript
// 修改前
hp: template.hp

// 修改後
hp: template.hp * 1.5  // 非主堡單位 +50%
```

**效果**：
- ✅ 戰鬥持續更久
- ✅ 更多回合交換
- ✅ 更有觀賞性

#### 提高技能頻率
```javascript
// 修改前
Math.random() < 0.25  // 25% 技能釋放率

// 修改後
Math.random() < 0.5   // 50% 技能釋放率
```

**效果**：
- ✅ 死靈術士更常召喚骷髏 💀
- ✅ 冰法師更常釋放冰霜新星 ❄️
- ✅ 所有技能更頻繁展示

---

### 4. 平局時間調整

```javascript
// 修改前
if (B.timer > 180000) { endBattle('draw'); }  // 3 分鐘

// 修改後
if (B.timer > 600000) { endBattle('draw'); }  // 10 分鐘
```

**原因**：
- 血量提高後戰鬥更持久
- 給玩家更多時間部署策略
- 避免平局過早

---

### 5. BOSS 系統調整

**BOSS 位置**：
```javascript
// 修改前：10×10 地圖
bossPosition: {x:7, y:2}  // 右側

// 修改後：9×7 地圖
bossPosition: {x:3, y:0}  // 上方中央
```

**小兵位置調整**：
- 暗影領主：左右兩側 (1,1) 和 (6,1)
- 烈焰龍王：左右兩側 + 更分散
- 寒冰泰坦：左右 + 中間
- 雷霆巨獸：左右 + 中間
- 虛空帝王：左右各 2 個（共 4 個）

**障礙物調整**：
- 所有障礙物位置適配 9×7 地圖
- 更合理的佈局

## 🐛 已知問題

### ⚠️ 普通關卡位置問題

**問題描述**：
- 100 個普通關卡的敵人位置仍是 10×10 地圖座標
- 敵人可能出現在邊界外或不合理位置

**影響範圍**：
- 所有普通關卡（1-1 到 10-10）
- BOSS 副本已修復 ✅

**解決方案**：
- 需要批量轉換所有關卡座標
- 座標轉換規則：
  - X：保持或縮小 (0-9 → 0-8)
  - Y：保持或縮小 (0-9 → 0-6)
  - 主堡周圍留空

**預計工作量**：
- 約 1-2 小時批量處理
