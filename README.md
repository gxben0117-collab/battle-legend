## 📖 簡介

**戰棋傳說**是一款融合了**自走棋**、**卡牌收集**和**戰術策略**的單機遊戲。玩家通過組建牌組、佈署單位、運用羈絆效果，在 9×9 的戰場上與敵人展開自動戰鬥。

### ✨ 核心特色

- 🎴 **物魔分離系統**：物理攻擊/魔法攻擊獨立計算，職業相剋明確
- 🏰 **靜態主堡**：聚焦單位戰鬥，去除微操複雜度
- 🗺️ **地形障礙**：每關獨特地形配置，增加策略深度
- ⚡ **羈絆系統**：種族/職業羈絆效果，鼓勵多樣化組牌
- 💎 **卡牌收集**：8 種職業、4 種稀有度，豐富的收集樂趣
- 📦 **單文件遊戲**：無需安裝，打開即玩

---

## 🎯 遊戲玩法

### 戰鬥系統

- **ATB 條系統**：每個單位有獨立的行動條，速度越高行動越快
- **自動戰鬥**：單位自動尋找目標並攻擊，玩家專注於策略佈局
- **技能系統**：每個單位有獨特主動技能和被動效果
- **能量系統**：每 2 秒回復 1 點能量，用於部署新單位

### 策略要素

1. **組牌策略**：30 張卡牌組合，平衡物理/魔法輸出
2. **羈絆搭配**：觸發種族/職業羈絆獲得強力增益
3. **開局部署**：選擇最多 3 張低費卡開局（總費用 ≤ 6）
4. **戰術選擇**：自由攻擊 / 集火最低血 / 突擊主堡

## 🃏 卡牌系統

### 稀有度分類

| 稀有度 | 標記 | 顏色 | 代表卡牌 |
|--------|------|------|---------|
| 普通 | N | 灰色 | 聖光騎士、狂野戰士、翠林弓手 |
| 稀有 | R | 藍色 | 星穹法師、影刃刺客 |
| 超稀有 | SR | 紫色 | 鐵壁護衛、神聖牧師 |
| 超超稀有 | SSR | 金色 | 死靈術士（英雄卡）|

### 職業分類

- **⚔️ 戰士**：高生命、近戰、物理輸出
- **🏹 射手**：遠程、物理、高機動
- **🔮 法師**：超遠程、魔法、範圍傷害
- **🗡️ 刺客**：高速、高暴擊、切後排
- **🛡️ 坦克**：超高防禦、保護友方
- **✨ 牧師**：治療、增益、輔助
- **💀 召喚師**：召喚單位、持續作戰

## 🌟 v0.2 新功能

### 物攻/魔攻/物防/魔防 四維屬性

```
物理傷害 = 物攻 - 目標物防 × 0.5
魔法傷害 = 魔攻 - 目標魔防 × 0.5
```

**職業相剋**：
- 法師（魔攻）→ 剋 → 坦克（低魔防）
- 坦克（高物防）→ 剋 → 戰士/射手（物理）
- 刺客（高魔防）→ 剋 → 法師（脆皮）

### 主堡系統

- 指揮官改為靜態主堡（🏰 我方 / 🚩 敵方）
- 永遠不移動，只有生命值
- 簡化戰鬥邏輯，聚焦單位策略

### 地形障礙系統

- 每關配置 3~5 個障礙物（🪨 岩石 / 🌳 樹木）
- 障礙格不可進入，單位自動繞行
- 增加戰場變化和策略深度

### 羈絆系統

**種族羈絆**：
- 👥 人類（2/4/6）：命中 +5%/+10%/+15%+暴擊
- 🐺 獸人（2/4）：生命 +10%/+20%+物攻
- 🧝 精靈（2/3）：速度 +10/+20+閃避
- 💀 亡靈（2/3）：魔攻 +15%/+30%+魔防
- 👼 天使（1/2）：全體魔防 +3/+6

**職業羈絆**：
- ⚔️ 戰士（2/4）：物防 +3/+6+生命
- 🏹 射手（2/3）：射程 +1/+1+物攻

### 戰鬥表現強化

- 💚 漸層血條（綠色健康漸層 + 高光效果）
- 💙 護盾條系統（藍色發光效果）
- ❤️ 低血閃爍（HP < 30% 紅色脈動）

### 牌組張數限制

- 一般單位卡：同名最多 **3 張**
- 英雄卡：同名最多 **1 張**

## 🎮 如何遊玩

### 在線遊玩（推薦）

👉 **[立即遊玩](https://yourusername.github.io/battle-legend)** 👈

### 本地遊玩

1. 下載 `index.html`
2. 用瀏覽器打開（推薦 Chrome / Edge / Firefox）
3. 開始遊戲！

### 操作說明

- **主選單**：開始新遊戲 / 繼續遊戲
- **主介面**：選擇功能（關卡/牌組/商店/生產）
- **戰鬥**：
  - 🎯 選擇戰術（自由/集火/突擊）
  - ⚡ 調整倍速（×1 / ×2 / ×4）
  - 能量足夠時自動打出手牌
