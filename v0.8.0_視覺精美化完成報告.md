## 🎯 技術細節

### CSS 技術運用

#### 1. 多層陰影（Box-shadow）
```css
box-shadow:
  0 0 10px 顏色A,    /* 內層發光 */
  0 0 20px 顏色B,    /* 中層擴散 */
  0 0 40px 顏色C,    /* 外層光暈 */
  0 4px 8px 陰影;    /* 立體陰影 */
```

#### 2. 文字發光（Text-shadow）
```css
text-shadow:
  0 0 10px 發光色,
  0 0 20px 擴散色,
  0 2px 4px 陰影色;
```

#### 3. 濾鏡特效（Filter）
```css
filter:
  drop-shadow(0 0 10px 顏色)
  drop-shadow(0 4px 8px 陰影);
```

#### 4. 漸變背景（Gradient）
```css
/* 線性漸變 */
background: linear-gradient(145deg,
  起始色 0%,
  中間色 50%,
  結束色 100%);

/* 徑向漸變（寶石質感） */
background: radial-gradient(circle at 30% 30%,
  亮色 0%,
  中色 50%,
  暗色 100%);
```

#### 5. 關鍵幀動畫（Keyframes）
```css
@keyframes 動畫名 {
  0%, 100% { /* 起始和結束狀態 */ }
  50% { /* 中間狀態 */ }
}

animation: 動畫名 2s ease-in-out infinite;
```

#### 6. 偽元素（Pseudo-elements）
```css
.element::before {
  content: '符號';
  position: absolute;
  /* 裝飾性內容 */
}

.element::after {
  content: '';
  /* 光效、標記 */
}
```
